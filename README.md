# 🎰 Multiplayer Bingo Game

A real-time multiplayer Bingo game with Telegram bot integration, wallet system, and admin dashboard.

## Features

- 🎮 Real-time multiplayer Bingo gameplay
- 🤖 Telegram bot for user registration
- 💰 Wallet system with deposit/withdraw
- 🏆 Winner announcements with animations
- 📊 Admin dashboard for management
- 📱 Mobile-friendly responsive design
- 🔔 Real-time WebSocket updates

## Tech Stack

- **Backend:** Node.js, Express, Socket.io, PostgreSQL
- **Frontend:** React.js, Socket.io-client
- **Admin Panel:** React.js, Recharts
- **Telegram Bot:** Telegraf
- **Database:** PostgreSQL with Sequelize ORM
